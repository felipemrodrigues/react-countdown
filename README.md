React Count To
==============

Animated countdown component for [React.js](http://facebook.github.io/react/)

Installation
------------

```sh
$ npm install react-countdown
```

Usage
-----

```javascript
import Counter from 'react-countdown';

<Countdown finalDate="" />

Attributes
----------

- **className** (optional): CSS class.
- **finalDate**: Delta of final date.

Test
----

```sh
$ npm test
```
